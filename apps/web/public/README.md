# Web Public Assets

Vite serves files in this folder from the site root.

Current root assets:

- `/logo.svg`
- `/favicon.svg`
- `/icons/icon-192.png`
- `/icons/icon-512.png`

Examples for organized assets:

- `public/assets/images/logo.png` becomes `/assets/images/logo.png`
- `public/assets/icons/favicon.svg` becomes `/assets/icons/favicon.svg`
- `public/assets/fonts/example.woff2` becomes `/assets/fonts/example.woff2`

Use this folder for static files that should keep their exact filename and URL. Put imported component assets in `src/` instead.
