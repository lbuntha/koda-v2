## Milestone 2 — Admin & Analytics (2 weeks)

Full admin UI (taxonomy, skills browser, users, tiers, locales, flags, reconcile, audit), parent analytics, reconciliation tools.

## Milestone 3 — Engagement (2 weeks)

Streak bonuses, daily goal, gems, mascot, skill tree, missions, push notifications.

## Milestone 4 — OCR + LLM (2 weeks)

OCR worksheet → questions pipeline. LLM-assisted skill authoring.

## Milestone 5 — Social (2 weeks)

Friends, class groups, multiplayer (Study Together).

## Milestone 6 — Monetization (1 week)

Stripe family tier, billing portal, tier enforcement.
