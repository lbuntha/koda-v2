# Kodaapp Reference Docs

These files were copied from the original `kodaapp` project so `koda-v2` can continue with the same product and design context.

Read in this order for new work:

1. `MVP_KICKOFF_CHECKLIST.md` - launch priorities and core user flow.
2. `ARCHITECTURE.md` - frontend feature organization from the original app.
3. `AI_RULES.md` - project rules, protected areas, role/platform expectations.
4. `DESIGN_STANDARDS.md` and `THEME.md` - Koda UI and theme guidance.
5. Placement and onboarding docs - parent/child onboarding behavior.

`koda-v2` is not a direct copy of `kodaapp`; it is the rebuild described in `../MILESTONES.md`. Treat these docs as product reference unless a milestone explicitly says to port code.
